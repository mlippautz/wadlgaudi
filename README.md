# Wadlgaudi Web

A privacy-first, decentralized activity tracker for sports like cycling, running, and skiing. Built using the AT Protocol (Bluesky).

## Features

- **Decentralized**: Your data lives on your PDS (Personal Data Server).
- **Privacy First**: Sensitive GPS data is end-to-end encrypted.
- **Social Discovery**: Find activities through your existing Bluesky network.
- **Modern UI**: High-performance Web Components with a premium dark theme.

## Development

### Setup

```bash
pnpm install
```

### Scripts

- `pnpm dev`: Start development server.
- `pnpm build`: Build for production.
- `pnpm test`: Run tests with Vitest.
